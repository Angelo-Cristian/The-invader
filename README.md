# 🎮 The Invader
Drive your flying ship while avoiding obstacles and fighting fearsome enemies in this 3D game made in Unity.

---

## 🛠️ Technical details
* **Movement system** -> The airship can move up (W), down (S), left (A), and right (D) on the screen, without leaving the boundaries, by using Unity's Input System.
* **Timeline/Animation system** -> A Unity Timeline handles the forward progression of the level, synchronizing the arrival of obstacles and enemy attack patterns to challenge the player.
* **Laser attacks** -> The airship can shoot lasers (Left Click / CTRL) to defend itself and destroy obstacles and enemies.
* **Score system** -> When you destroy an enemy or an obstacle, your score increases.
* **Death system** -> If the airship collides with an obstacle or an enemy, it gets destroyed and it's game over.
* ⚠️ *Note: I do not own the airship models, trees, or any textures used in this project.*
